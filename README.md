# College GPA Calculator

A clean, responsive web app for calculating semester and cumulative GPA on a standard 4.0 scale.

## Features

- **Semester GPA** — Add multiple courses with credit hours and letter grades to calculate your semester GPA
- **Cumulative GPA** — Enter your previous GPA and credit hours to compute your overall cumulative GPA
- **Dynamic course rows** — Add or remove courses as needed (minimum one row)
- **Real-time calculation** — GPA updates instantly as you enter data
- **Input validation** — Invalid credit values are highlighted
- **Responsive design** — Works on desktop and mobile

## Grade Scale

| Grade | Points |
|-------|--------|
| A     | 4.0    |
| A-    | 3.7    |
| B+    | 3.3    |
| B     | 3.0    |
| B-    | 2.7    |
| C+    | 2.3    |
| C     | 2.0    |
| C-    | 1.7    |
| D+    | 1.3    |
| D     | 1.0    |
| F     | 0.0    |

## Usage

Open `index.html` in any modern browser. No build tools or dependencies required.
